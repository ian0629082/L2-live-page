# L2-DIC1 - Ian's Live DateTime Landing Page

A premium, interactive personal landing page featuring a live date/time display with styled ambient effects.

🔗 **Live Demo**: https://ian0629082.github.io/L2-live-page/

## 🚀 Features

- **Live Digital Clock**: Updates in real-time with seconds.
- **Dynamic Greetings**: Changes automatically based on the time of day (Good morning, Good afternoon, Good evening, Good night).
- **Customized Date Layout**: Formatted dynamically with weekday names and ordinal suffixes (e.g. `May 29th, 2026`).
- **Interactive Time Format**: Easily toggle between 12-hour (AM/PM) and 24-hour time formats.
- **Stunning Glassmorphism Design**: Glow background circles, high-contrast dark aesthetics, and micro-animations for responsive user interaction.

## 📁 Project Structure

- **`index.html`**: The page layout and markup structure, configured with SEO tags.
- **`index.css`**: Core styling, responsive typography (Google Fonts Outfit & Inter), glowing ambient keyframe animations, and custom CSS variables.
- **`app.js`**: Pure JavaScript logic handling clock ticks, date suffix parsing, and format switching.

## 💻 How to Run Locally

Simply clone the repository and open `index.html` in your browser, or spin up a quick server in the project directory:

```bash
# Using Node.js http-server
npx http-server

# Or python server
python -m http.server 8080
```
